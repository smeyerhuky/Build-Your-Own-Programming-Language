# H2 2026 Hiring Plan

*Three hires are planned for H2 2026 (post-pre-seed close): Founding Engineer, Head of Developer Relations, and a CEO Co-Founder (separate process — see [team.md](../company/team.md)).*

---

## Hire 1 — Founding Engineer (Compiler/Platform)

**Priority:** P0  
**Target start:** Q4 2026 (October/November)  
**Compensation target:** $130K–$160K base + 0.5–1.0 % equity (4-yr vest, 1-yr cliff)  
**Reports to:** Clinton L. Jeffery (CTO)

### Role summary

The founding engineer will build and maintain the core LangCraft platform: the cloud sandbox, exercise infrastructure, LLM integration, and the web front-end. This person will own the full stack from the J0 compiler containerisation to the React UI.

### Required qualifications

- 5+ years of software engineering experience
- Strong Java or Kotlin (the J0 compiler is Java/Unicon; backend will be Java or JVM-based)
- Experience building and deploying containerised applications (Docker, Kubernetes or ECS)
- Comfortable with React or equivalent modern front-end framework
- Genuine interest in compilers, programming languages, or systems software

### Strongly preferred

- Has built a compiler or interpreter before (even a toy one)
- Experience with language server protocol (LSP) or tree-sitter
- Experience with LLM API integration (OpenAI, Anthropic)
- Open-source contributions visible on GitHub

### Red flags

- Has never read compiler error messages with genuine curiosity
- Expects a fully-defined spec before writing any code
- Not interested in or familiar with the J0 language / course content

### Interview process

1. **30-min screen** (Clinton): background, motivation, culture fit
2. **Take-home exercise** (4 hours): "Fix these 3 J0 compiler bugs and write a test for each"
3. **Technical interview** (2 hrs, Clinton + one external): architecture design + code walkthrough of take-home
4. **Reference calls** (2): previous manager or technical colleague
5. **Offer**

### Job posting channels

- GitHub Jobs
- Hacker News "Who is Hiring" thread (posted monthly)
- LinkedIn (Clinton's network)
- University CS department job boards (New Mexico Tech, UNM, NM State)

## Hire 2 — Head of Developer Relations

**Priority:** P0  
**Target start:** Q4 2026 (November/December)  
**Compensation target:** $120K–$140K base + 0.25–0.5 % equity (4-yr vest, 1-yr cliff)  
**Reports to:** CEO (once hired); Clinton in the interim

### Role summary

The Head of DevRel will own the developer community, content, and advocacy function. This is a generalist role in a very early startup: part content creator, part community manager, part technical evangelist, part conference speaker.

### Required qualifications

- 4+ years of developer relations, developer advocacy, or technical marketing
- Proven track record of building a developer community from scratch (not just maintaining an existing one)
- Strong technical writing skills: can write a 2 000-word tutorial that developers actually finish
- Has given conference talks or produced YouTube content that got real engagement
- Genuine interest in compilers, programming languages, or developer education

### Strongly preferred

- Has grown a technical OSS project community (GitHub stars, Discord, forum)
- Experience with SEO and organic content marketing for developers
- Has worked at a developer tools company (GitHub, JetBrains, HashiCorp, Stripe, etc.)
- Has an existing audience (Twitter/X, newsletter, YouTube) of developers

### Red flags

- Thinks "developer relations" means writing marketing copy
- Has never shipped code themselves
- Primary experience is with non-technical marketing

### Interview process

1. **30-min screen** (Clinton): background, motivation, sample work review
2. **Work sample submission** (3 days): "Write a 1 000-word tutorial on how J0 handles type checking. Include at least one runnable code example."
3. **Campaign design exercise** (1 hr, live): "Design a 30-day content plan for LangCraft's launch"
4. **Reference calls** (2)
5. **Offer**

## Hire 3 — Sales Engineer (Enterprise)

**Priority:** P1  
**Target start:** Q1 2027 (after launch traction validated)  
**Compensation target:** $120K–$150K base + $30K–$50K variable + 0.2–0.4 % equity  
**Reports to:** CEO

### Role summary

The Sales Engineer will own the technical side of the enterprise sales motion: demos, proof-of-concept engagements, security questionnaires, and technical integration support. They are the bridge between the product and enterprise buyers.

### Required qualifications

- 4+ years of sales engineering, solutions architecture, or technical account management
- Strong technical skills: can read Java code, understand compiler concepts at a high level, navigate CLI tools
- Experience with enterprise software procurement: MSA, DPA, security questionnaires, SOC 2
- Experience with developer tools or devops tools sales cycles

### Strongly preferred

- Has worked at a developer infrastructure company (HashiCorp, Atlassian, JetBrains, GitLab, etc.)
- Has closed enterprise deals >$50K ACV
- Understands SSO (SAML/OIDC) and enterprise security requirements
