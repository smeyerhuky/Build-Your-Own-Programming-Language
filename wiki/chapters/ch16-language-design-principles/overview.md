# Overview

Themes the book closes on:

- **Tooling** — a debugger, a profiler, a package manager, an IDE
  plugin. Most of these are work projects in their own right.
- **Standard library** — even Hello World needs `println`; a useful
  language needs collections, file I/O, networking.
- **Compatibility story** — versioning, deprecation policy,
  migration tooling.
- **Performance** — register allocation, inlining, escape analysis,
  GC tuning.
