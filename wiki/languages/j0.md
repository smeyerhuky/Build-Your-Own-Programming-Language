# J0

J0 is a deliberately small Java subset, chosen so a single book can
build a complete compiler for it. A J0 program is one top-level class
with fields, methods, and a `public static void main` entry point.

## Reserved words (from `/ch3/javalex.l`)

```
break  bool   class  double  else  false  for  if  int
null   public return static  string  true  void  while
```

## Operators

```
+  -  *  /  %        arithmetic
=  +=  -=            assignment
==  !=  <  <=  >  >= comparison
&&  ||  !            logical
.  ,  ;  :           punctuation
(  )  {  }  [  ]     grouping
```

## Types

- `int`, `double`, `bool`, `string`, `void`.
- Array types: `T []`, `T [][]`, …
- Class types: user-defined.

## Statements

- Local var decl: `Type name [= expr];`
- `if (e) S` / `if (e) S else S` / `if-else-if` chains.
- `while (e) S`, `for (init; cond; update) S`.
- `break;`, `break L;`
- `return;`, `return e;`
- Expression statements: assignment, method call, instantiation.

## Built-ins

The Ch 6 symbol table installs a synthetic `System` class with an
`out` field and `println()` method so `System.out.println("hi");`
parses and runs without a real standard library.

## Grammar entry point

`/ch4/j0gram.y` starts at `ClassDecl: PUBLIC CLASS IDENTIFIER ClassBody;`.

## What J0 deliberately omits

- Generics.
- Interfaces, abstract classes, modifiers other than `public static`.
- Exceptions.
- Reflection.
- A real standard library.
- Implicit numeric widening on assignment (allowed in expressions only).

## Hello, world

```java
public class hello {
  public static void main(string[] argv) {
    System.out.println("hello, world");
  }
}
```

This is the program in `/ch3/hello.java` and every later chapter's
`hello.java`.
