# Prompt patterns

Copy-paste templates for asking an LLM (or GitHub Copilot) to help
at each stage of setup. Each template is structured as a context
block + question so the model has enough information to give a useful
answer.

For deeper prompting theory see
[Wiki: prompting](/wiki/prompting/index.md).

- [Setup assistant](./setup-assistant.md) — install and verify failures.
- [First-error debug](./first-error-debug.md) — compiler build failures.
- [Extend J0](./extend-j0.md) — adding lexer / grammar rules.
