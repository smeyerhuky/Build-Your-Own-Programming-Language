# Tools

Quick-ref cards focused on installation, verification, and the most
common failure modes for each tool. For deeper tool internals see the
wiki's tool pages at [`/wiki/tools/`](/wiki/tools/index.md).

- [JDK](./jdk.md) — Java compiler + runtime.
- [JFlex](./jflex.md) — Lexer generator (`.l` → `Yylex.java`).
- [BYacc / CUP](./byacc-cup.md) — Parser generator (`.y` → `parser.java`).
- [Make](./make.md) — Build automation.
- [GCC / Clang](./gcc.md) — C compiler for Ch 13 native output.
- [Unicon](./unicon.md) — Optional: builds the `.icn` variant chapters.
